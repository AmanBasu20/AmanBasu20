## 💼 Featured Projects

### [Credit Card Fraud Detection Pipeline](https://github.com/AmanBasu20/credit-card-fraud-detection)
End-to-end fraud detection pipeline on severely imbalanced transaction data (0.172% fraud rate),
using SMOTE and strict train-only preprocessing to eliminate data leakage.
- Benchmarked Logistic Regression, MLP, and XGBoost on a 56,961-sample stratified test set
- Selected the final model using AUPRC (0.8594) and fraud recall (88.8%) after systematic cross-validation
- Applied precision-recall-based evaluation suited for high-stakes, imbalanced classification problems
- Tech: Python · XGBoost · TensorFlow · scikit-learn

### [Amazon Electronics Recommendation Engine](https://github.com/AmanBasu20/amazon-electronics-recommender)
Unsupervised recommendation engine using collaborative filtering and Truncated SVD to extract
latent purchasing patterns from a large, sparse user-item matrix.
- Implemented cosine-similarity retrieval in latent feature space
- Reduced inference latency through offline similarity pre-computation and artifact caching
- Deployed as an interactive Streamlit application
- Tech: Python · scikit-learn · Pandas · Streamlit
- Live Demo: https://amazon-electronics-recommender-5a4zribrwpwxsippw5qcg7.streamlit.app/

### [Employee Salary Prediction Web App](https://github.com/AmanBasu20/employee-salary-prediction)
Full ML pipeline on the UCI Adult Income dataset, built during the IBM SkillsBuild internship,
predicting income class from demographic and employment features.
- Performed EDA, feature engineering, and categorical encoding
- Evaluated Logistic Regression, KNN, MLP, Random Forest, and XGBoost via cross-validation
- Achieved 87.5% accuracy with XGBoost; deployed as a real-time Streamlit app
- Tech: Python · XGBoost · scikit-learn · Streamlit
- Live Demo: https://employee-salary-prediction-amanbasu20.streamlit.app/

### [Mandi-Alert](https://github.com/AmanBasu20/Mandi-Alert)
Real-time mandi price monitoring and alert system that tracks commodity market movements and
helps users identify important price changes.
- Built to support timely decision-making using market data
- Designed for practical commodity price tracking and alert generation
- Tech: Python · Pandas · Streamlit · SQL
